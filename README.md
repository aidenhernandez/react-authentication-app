# React-Authentication-App

## Get started

Install the dependencies for Front End and Backend..

FrontEnd
```bash
cd front-end
npm install
```

...then start:

```bash
npm run start
```

Backend
```bash
cd back-end
npm install
```

...then start:

```bash
npm run dev
```

Navigate to [localhost:8080](http://localhost:8080) for backend and to [localhost:3000](http://localhost:3000) for frontend.
